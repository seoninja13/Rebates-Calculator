# Property Naming Requirements

1. Raw Data Format:
- Use unprefixed names (programName, programType, etc.)
- Must include all required fields
- Follow camelCase naming convention

2. Transformed Data Format:
- ALL properties must use trans_ prefix
- Example: programName becomes trans_title
- Maintain consistent naming across all transformations

3. Required Property Mappings:
programName -> trans_title
programType -> trans_type
summary -> trans_summary
collapsedSummary -> trans_collapsedSummary
amount -> trans_amount
eligibleProjects -> trans_eligibleProjects
eligibleRecipients -> trans_eligibleRecipients
geographicScope -> trans_geographicScope
requirements -> trans_requirements
applicationProcess -> trans_applicationProcess
deadline -> trans_deadline
websiteLink -> trans_websiteLink
contactInfo -> trans_contactInfo
processingTime -> trans_processingTime
category -> trans_category
